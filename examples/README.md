# Exemplos

Esta pasta contem dados ficticios para demonstrar a estrutura esperada pelo motor de escalas.

## Ficheiros

- `estado_exemplo.json`: exemplo de configuracao com funcoes, colaboradores, horarios, necessidades e regras.

Para testar manualmente, copie o conteudo para um ficheiro local chamado `estado_escala.json`. Esse ficheiro local continua fora do Git para evitar publicar dados operacionais reais.
